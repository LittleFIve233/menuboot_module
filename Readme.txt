Taiko3U研究日志2023/8/28
起初以7月份，有人问能不能Wii U自启动游戏开始，以Wii U基板打造太鼓达人机台
在对AutobootModule小部分研究后，--我搓出来了AutobootModule汉化版--
此时我觉得超出了我的能力范围，于是果断选择停止研究
在8月中旬，我与一位其它群友搞NS时闲着没事尝试寻找现有的自制程序魔改为自启动模块
无意中看到了一个很基础的自制程序，于是尝试从这个自制程序入手，手搓自启动模块
在临近11点左右，模块手搓成功，诞生出第一个版本（以旷野之息为测试的版本）
很快问题发生了。。
旷野之息必须要更新档，这个补丁无法运行更新档
但是在太鼓达人就用不着多想了，本身就是为太鼓达人研究的模块
这个消息被我公布到Wii UHombrew群聊中，在群聊里聊天的时候，突然想到能不能把它插入自启动模块行列
事实是可以的，不用二次加载，开机后直接进入太鼓达人
次日我将它命名为menubootModule
到此为第一部分，基础模块的研究
我尝试与Aroma结合，成功！
这就说明了可以把SDcafiine引入，扩展性则高了不少倍
在整理完太鼓达人Wii U1、2作的DLC后，我试着放入3里运行
结果是我的思路完全正确，正常运行，但是随之而来的问题是，安装的DLC不能被加载
此为第二部分，引入12作的DLC
在月末，我试想删掉安装的DLC完全用挂载，没成想没有AOC分区SDcafiine完全不能跑
于是我就想制作一个虚假DLC，其内容可以完全用sdcafiine加载
这个虚假DLC做起来也很简单，只需要原版DLC搬空Content内容即可
350kb的DLCReloader就这样做出来了
剩下的只有整合123的DLC了
